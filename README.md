# DiffAI 🧠✨
**AI-powered Git assistant that understands your diffs.**

DiffAI helps developers make better commits, generate meaningful PR summaries, and automate changelogs — all powered by AI that *actually understands code changes*.

---

## 🚀 Features
- 🤖 Generate **commit messages** from staged diffs  
- 📑 Summarize **pull requests** into developer-friendly overviews  
- 📝 Auto-generate **changelogs** grouped by features & fixes  
- 🔌 Works with Git CLI (`git diffai commit`)  
- 🌍 Extensible for GitHub, GitLab, Bitbucket integrations

## 🔹 4. Repo Settings
When creating the GitHub repo:  

- **Visibility** → Public (so it’s open-source right away).  
- **License** → MIT (developer-friendly, allows wide adoption).  
- **Issues** → Enabled (default, for bug reports & feature requests).  
- **Discussions** → Enable (for brainstorming & community feedback).  
- **Actions** → Enable (for CI/CD, later AI integration checks).  
- **Topics/Tags** → `git`, `cli`, `ai`, `commit-messages`, `devtools`, `opensource`.
